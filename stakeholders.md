# Stakeholders — Study Planner

**Коротко**
- **Користувачі (End users):** студенти — основні отримувачі цінності.
- **Оцінювачі (Instructors):** оцінюють виконання, можуть дати зворотний зв'язок.
- **Розробники (Dev team/You):** відповідають за реалізацію.
- **Дизайнери/Ментори (опціонально):** підказують по UX/USABILITY.

**Очікування та обмеження**
- Користувачі: простота, швидкість, зрозумілість.
- Оцінювачі: наявність README з пітчем і критеріями, простий demo.
- Розробники: короткий час на реалізацію MVP, мінімальний набір тестів.
