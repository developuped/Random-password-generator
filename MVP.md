# MVP — Study Planner (детально)

**Що входить в MVP**
- Додавання задачі: мінімум — текстова назва; опціонально — термін.
- Перегляд списку задач: відображення назви та статусу.
- Позначення задачі як виконаної (toggle).
- Прості acceptance тест-кейси (ручні).

**Пріоритет функцій**
1. Додавання задачі (високий)
2. Відображення списку (високий)
3. Позначення як виконаної (високий)
4. Збереження локально (середній)
5. UI-поліпшення (низький)

**Пропозиції для реалізації**
- Web: HTML + JS, збереження у localStorage.
- Mobile/web hybrid: простий PWA для локального збереження.
